# 两个线程交替打印12345和ABCDE

## 使用wait、notify

```java
public class Main {
    private static int num = 1;
    private static char ch = 'A';
    private static final Object lock = new Object();

    public static void main(String[] args) {
        Runnable runnable = () -> {
            synchronized (lock) {
                for (int i = 0; i < 5; i++) {
                    if (Thread.currentThread().getName().equals("numThread")) {
                        //打印数字
                        System.out.println((num++));
                        //唤醒其他等待的线程
                        lock.notifyAll();
                        // 让当前线程释放锁资源，进入wait状态
                        try {
                            lock.wait();
                        } catch (InterruptedException e) {
                            e.printStackTrace();
                        }
                    } else if (Thread.currentThread().getName().equals("charThread")) {
                        System.out.println(ch++);
                        //唤醒其他等待的线程
                        lock.notifyAll();
                        // 让当前线程释放锁资源，进入wait状态
                        try {
                            lock.wait();
                        } catch (InterruptedException e) {
                            e.printStackTrace();
                        }
                    }
                }
                lock.notifyAll();
            }
        };
        new Thread(runnable,"numThread").start();
        new Thread(runnable,"charThread").start();
    }
}
```

## 使用condition条件唤醒

```java
import java.util.concurrent.locks.Condition;
import java.util.concurrent.locks.Lock;
import java.util.concurrent.locks.ReentrantLock;

public class Main {
    private static Lock lock = new ReentrantLock();
    private static Condition condition1 = lock.newCondition();
    private static Condition condition2 = lock.newCondition();
    private static int state = 0;
    private static int index = 1;
    private static char ch = 'A';


    private static void print(int times, int currentState, Condition cur, Condition next) {
        for (int i = 0; i < times; i++) {
            lock.lock();
            try {
                if (state % 2 != currentState) {
                    cur.await();
                }
                if (currentState == 0) {
                    System.out.println(index++);
                } else {
                    System.out.println(ch++);
                }
                state++;
                next.signal();
            } catch (Exception e) {

            } finally {
                lock.unlock();
            }
        }
    }

    public static void main(String[] args) {
        new Thread(() -> print(5, 0, condition1, condition2), "numThread").start();
        new Thread(() -> print(5, 1, condition2, condition1), "charThread").start();
    }
}
```

# 两个线程交替打印奇数和偶数

```java
import java.util.concurrent.TimeUnit;

public class Main {
    private static final Object lock = new Object();
    private static int i = 0;

    public static void main(String[] args) throws InterruptedException {
        Runnable runnable = () -> {
            synchronized (lock) {
                while (i <= 100) {
                    try {
                        System.out.println(Thread.currentThread().getName() + ":" + i++);
                        lock.notifyAll();
                        lock.wait();
                    } catch (InterruptedException e) {
                        e.printStackTrace();
                    }
                }
                //防止有子线程被阻塞未被唤醒，导致主线程不退出
                lock.notifyAll();
            }
        };
        new Thread(runnable, "evenThread").start();
        TimeUnit.MILLISECONDS.sleep(10);
        new Thread(runnable, "oddThread").start();
    }
}
```

# 三个线程分别打印 A，B，C

## 使用condition条件唤醒

```java
import java.util.concurrent.locks.Condition;
import java.util.concurrent.locks.Lock;
import java.util.concurrent.locks.ReentrantLock;

public class Main {
    private static int state = 0;
    private static Lock lock = new ReentrantLock();
    private static Condition condition1 = lock.newCondition();
    private static Condition condition2 = lock.newCondition();
    private static Condition condition3 = lock.newCondition();

    private static void print(int targetState, Condition current, Condition next) {
        for (int i = 0; i < 10; i++) {
            lock.lock();
            try {
                while (state % 3 != targetState) {
                    current.await();
                }
                state++;
                System.out.println(Thread.currentThread().getName());
                next.signal();
            } catch (Exception e) {
                e.printStackTrace();
            } finally {
                lock.unlock();
            }
        }
    }

    public static void main(String[] args) {
        new Thread(() -> print(0, condition1, condition2), "A").start();
        new Thread(() -> print(1, condition2, condition3), "B").start();
        new Thread(() -> print(2, condition3, condition1), "C").start();
    }
}
```
